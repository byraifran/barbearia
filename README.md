## ✨ Funcionalidades

- **Design Premium & Imersivo:** Paleta de cores escura (Dark Mode) com contrastes em tons quentes, ideal para o público-alvo.
- **Navegação Fluida:** Menu superior rápido para transições eficientes entre as secções (`Início`, `Serviços`, `Contactos`).
- **Secção "Sobre a Barbearia":** Espaço dedicado à história, missão e valores da marca fundada em 2019.
- **Lista de Benefícios:** Destaque visual com ícones para os diferenciais do negócio (Atendimento, Espaço, Localização, Profissionais).
- **Cards de Serviços:** Exibição clara e organizada dos serviços oferecidos (Cabelo, Barba, Combo) com os respetivos preços e ícones minimalistas.
- **Rodapé Informativo:** Direitos autorais e identidade visual preservada no fecho da página.

## 🎨 Paleta de Cores (Identidade Visual)

O projeto utiliza uma combinação de cores que transmite sofisticação e modernidade:

- **Fundo Principal:** `#1a2026` (Cinza Escuro / Slate)
- **Destaques & Títulos:** `#d35400` / `#e67e22` (Laranja Terracota / Amber)
- **Texto Principal:** `#ffffff` (Branco)
- **Texto Secundário:** `#b2bec3` (Cinza Claro)
